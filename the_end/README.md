XLP 的目标是构建一个可持续运营的群体学习系统，在该系统中可以创建众多不同内容的学习情境，人才团队根据任务内容灵活重组，通过群体学习，成为高绩效团队。最终，随着参加到群体学习环境的人数增多，我们可以预见：每个生态子系统都越来越多地模仿现实世界的行为模式，伴随着世界各地的人员、组织、学术机构、系统和社会参与其中，人与组织通过学习日趋完善 ―― 与此同时，也让我们的世界变得更美好。
